# Java Console Agenda

Proyecto de práctica personal desarrollado para repasar y afianzar las bases de Java
orientado a backend.

## Descripción
Aplicación de consola que simula una agenda de contactos.  
El objetivo principal del proyecto es trabajar la programación orientada a objetos,
la separación de responsabilidades y la lógica de negocio básica, sin interfaz gráfica.

## Objetivos del proyecto
- Repasar Java tras un tiempo sin utilizarlo
- Aplicar conceptos de programación orientada a objetos
- Estructurar clases y métodos de forma clara
- Practicar lógica backend y validaciones básicas
- Sentar una base para una posible evolución a API REST en el futuro

## Funcionalidades previstas
- Añadir contactos
- Listar contactos
- Buscar contactos
- Eliminar contactos
- Validar datos básicos (email, teléfono, duplicados)

## Tecnologías
- Java
- Programación en consola
- Git y GitHub
